    description

    variable name

    \(d^{x}_{s}(i)\)

    degrees of freedom for signal per level

    <x>_dfs {:,vertical}

    \(A^{x}(i,j)\)

    AVK of a profile x

    <x>_avk {:,vertical,vertical}

    The pattern : for the first dimensions can represent {latitude,longitude}, {time}, {time,latitude,longitude}, or no dimensions at all.

    \[d^{x}_{s}(i) = A^{x}(i,i)\]
