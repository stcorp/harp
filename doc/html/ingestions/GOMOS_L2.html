Ingestion options

The table below lists the available ingestion options for GOMOS_L2 products.

option name legal values description
temperature local, model can be either ‘local’ or ‘model’; ‘local’ (the default) will retrieve the locally measured temperature and ‘model’ will return the temperature from the external model
air local, model can be either ‘local’ or ‘model’; ‘local’ (the default) will retrieve the locally measured air density and ‘model’ will return the air density from the external model

Mapping description

GOMOS Level 2 products only contain a single profile; all measured profile points will be provided in reverse order (from low altitude to high altitude) in the profile
