Ingestion options

The table below lists the available ingestion options for GOMOS_L2 products.

option name

legal values

description

temperature

model

retrieve the locally measured temperature (default) or the temperature from the external model (temperature=model)

air

model

retrieve the locally measured air density (default) or the air density from the external model (air=model)

Mapping description

GOMOS Level 2 products only contain a single profile; all measured profile points will be provided in reverse order (from low altitude to high altitude) in the profile
