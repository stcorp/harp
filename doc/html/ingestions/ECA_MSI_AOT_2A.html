Ingestion options

The table below lists the available ingestion options for ECA_MSI_AOT_2A products.

option name

legal values

description

angstrom

670/865

wavelength combination for which the angstrom exponent is extracted: 355/670 (default), or 670/865 (angstrom=670/865)

aot

865

wavelength for which to ingest the aerosol optical thickness: 670nm (default) or 865nm (aot=865)
