ACE_FTS_L2_iso

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of ACE_FTS_L2_iso data.

field name type dimensions unit description
datetime double {vertical} seconds since 2000-01-01 date and time of occultation 30 km geometric tangent point (in seconds since 2000-01-01 00:00:00)
latitude double {vertical} degree_north latitude of 30 km geometric tangent point for occultation
longitude double {vertical} degree_east longitude of 30 km geometric tangent point for occultation
altitude double {vertical} km tangent altitude grid for retrieved parameters and species
temperature double {vertical} K temperature
pressure double {vertical} hPa pressure
density double {vertical} cm^-3 atmospheric density
H2O (181)_volume_mixing_ratio double {vertical} ppmv volume mixing ratio for H2O (181)
H2O (181)_volume_mixing_ratio_uncertainty double {vertical} ppmv volume mixing ratio uncertainty for H2O (181). If this value is -888 the vmr was not retrieved but obtained by scaling the a priori value
H2O (171)_volume_mixing_ratio double {vertical} ppmv volume mixing ratio for H2O (171)
H2O (171)_volume_mixing_ratio_uncertainty double {vertical} ppmv volume mixing ratio uncertainty for H2O (171). If this value is -888 the vmr was not retrieved but obtained by scaling the a priori value
H2O (162)_volume_mixing_ratio double {vertical} ppmv volume mixing ratio for H2O (162)
H2O (162)_volume_mixing_ratio_uncertainty double {vertical} ppmv volume mixing ratio uncertainty for H2O (162). If this value is -888 the vmr was not retrieved but obtained by scaling the a priori value
CH4 (311)_volume_mixing_ratio double {vertical} ppmv volume mixing ratio for CH4 (311)
CH4 (311)_volume_mixing_ratio_uncertainty double {vertical} ppmv volume mixing ratio uncertainty for CH4 (311). If this value is -888 the vmr was not retrieved but obtained by scaling the a priori value
CH4 (212)_volume_mixing_ratio double {vertical} ppmv volume mixing ratio for CH4 (212)
CH4 (212)_volume_mixing_ratio_uncertainty double {vertical} ppmv volume mixing ratio uncertainty for CH4 (212). If this value is -888 the vmr was not retrieved but obtained by scaling the a priori value
