Ingestion options

The table below lists the available ingestion options for S5P_L2_AER_LH products.

option name

legal values

description

aerosol_pressure

unclipped

ingest the aerosol_mid_pressure that is clipped to the surface pressure (default) or the unclipped variant (aerosol_pressure=unclipped)

surface_albedo

772

whether to ingest the surface albedo at 758nm (default) or the surface alebedo at 772nm (surface_albedo=772); this option only has an effect for processor version >= 02.06.00 (for older products always the albedo at 758nm is ingested)
