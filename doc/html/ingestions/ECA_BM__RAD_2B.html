Ingestion options

The table below lists the available ingestion options for ECA_BM__RAD_2B products.

option name

legal values

description

direction

fore, aft

viewing direction: nadir (default), fore (direction=fore), aft (direction=aft)

edge_coordinate

one_weight

edge coordinate: zero weight (default), one weight (edge_coordinate=one_weight)

radiance

SW_MSI, SW_filtered, LW, LW_filtered

radiance: SW (default), SW from MSI (radiance=SW_MSI), SW filtered (radiance=SW_filtered), LW (radiance=LW), LW filtered (radiance=LW_filtered)

resolution

small, full, assessment

resolution: standard (default), small (resolution=small), full (resolution=full), or assessment (resolution=assessment)
