Ingestion options

The table below lists the available ingestion options for GOME_L2_ERSNTO products.

option name legal values description
detailed_results BrO, H2O, HCHO, NO2, O3, OClO, SO2 include additional detailed results for the given species
corrected true include corrected VCD and/or ESC (corrected=true) or uncorrected VCD/ESC (default); this only applies to the species for which additional detailed results are ingested (detailed_results is set to one of BrO, H2O, HCHO, NO2, O3, or OClO)
