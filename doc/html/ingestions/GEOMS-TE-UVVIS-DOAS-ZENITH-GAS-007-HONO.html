GEOMS-TE-UVVIS-DOAS-ZENITH-GAS-007-HONO

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of GEOMS-TE-UVVIS-DOAS-ZENITH-GAS-007-HONO data.

field name type dimensions unit description
instrument_name string     name of the instrument
site_name string     name of the site at which the instrument is located
datetime double {time} days since 2000-01-01 mean time of the measurement
datetime_start double {time} days since 2000-01-01 start time of the measurement
datetime_stop double {time} days since 2000-01-01 stop time of the measurement
instrument_latitude double   degree_north latitude of the instrument
instrument_longitude double   degree_east longitude of the instrument
instrument_altitude double   m altitude of the instrument relative to the location site
altitude double {time, vertical} km effictive retrieval altitude
pressure double {time, vertical} hPa independent pressure profile
temperature double {time, vertical} K independent temperature profile
altitude_bounds double {time, vertical, 2} km lower and upper boundaries of the height layers
surface_wind_direction double {time} degree Wind direction at the station using WMO definition (wind from the north is 360; from the east is 90 and so on. No wind (calm) is 0)
surface_wind_speed double {time} m/s Wind speed at the station
solar_zenith_angle double {time} degree solar astronomical zenith angle
solar_azimuth_angle double {time} degree solar azimuth angle
viewing_azimuth_angle double {time} degree viewing azimuth angle of the instrument
viewing_zenith_angle double {time} degree viewing zenith angle of the instrument
latitude double {time, vertical} degree_north latitude of effective air mass at each altitude
longitude double {time, vertical} degree_east longitude of effective air mass at each altitude
cloud_flag string {time}   one of clear-sky, thin-clouds, thick-clouds, broken-clouds, unavailable
stratospheric_aerosol_optical_depth double {time} 1 total stratospheric aerosol optical depth user for the retrieval
HNO2_volume_mixing_ratio double {time, vertical} 1e6 ppmv HNO2 volume mixing ratio
HNO2_volume_mixing_ratio_cov_random double {time, vertical, vertical} 1e6 ppmv random covariance of the HNO2 volume mixing ratio
HNO2_volume_mixing_ratio_cov_systematic double {time, vertical, vertical} 1e6 ppmv systematic covariance of the HNO2 volume mixing ratio
HNO2_volume_mixing_ratio_apriori double {time, vertical} 1e6 ppmv a priori HNO2 volume mixing ratio
HNO2_volume_mixing_ratio_avk double {time, vertical, vertical} 1 averaging kernel for the HNO2 volume mixing ratio
tropospheric_HNO2_column_number_density double {time} Pmolec cm-2 tropospheric HNO2 column number density
tropospheric_HNO2_column_number_density_uncertainty_random double {time, vertical} Pmolec cm-2 random uncertainty of the tropospheric HNO2 column number density
tropospheric_HNO2_column_number_density_uncertainty_systematic double {time, vertical} Pmolec cm-2 systematic covariance of the tropospheric HNO2 column number density
tropospheric_HNO2_column_number_density_apriori double {time} Pmolec cm-2 a priori tropospheric HNO2 column number density
tropospheric_HNO2_column_number_density_avk double {time, vertical} 1 averaging kernel for the tropospheric HNO2 column number density
stratospheric_HNO2_column_number_density double {time} Pmolec cm-2 stratospheric HNO2 column number density
stratospheric_HNO2_column_number_density_uncertainty_random double {time, vertical} Pmolec cm-2 random uncertainty of the stratospheric HNO2 column number density
stratospheric_HNO2_column_number_density_uncertainty_systematic double {time, vertical} Pmolec cm-2 systematic covariance of the stratospheric HNO2 column number density
stratospheric_HNO2_column_number_density_apriori double {time} Pmolec cm-2 a priori stratospheric HNO2 column number density
stratospheric_HNO2_column_number_density_avk double {time, vertical} 1 averaging kernel for the stratospheric HNO2 column number density
stratospheric_HNO2_column_number_density_amf double {time} 1 air mass factor for the stratospheric HNO2 column number density
HNO2_column_number_density double {time, vertical} Pmolec cm-2 HNO2 partial column number density profile
HNO2_column_number_density_apriori double {time, vertical} Pmolec cm-2 a priori HNO2 partial column number density profile
index int32 {time}   zero-based index of the sample within the source product
