TES_L2_CO2_Nadir

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of TES_L2_CO2_Nadir data.

field name type dimensions unit description
datetime double {time} [seconds since 2000-01-01] time of the measurement
longitude double {time} [degree_east] tangent longitude
latitude double {time} [degree_north] tangent latitude
altitude double {time, vertical} [m] altitude per profile level
pressure double {time, vertical} [hPa] pressure per profile level
CO2_volume_mixing_ratio double {time, vertical} [ppv] carbon dioxide volume mixing ratio
CO2_volume_mixing_ratio_uncertainty double {time, vertical} [ppv] carbon dioxide volume mixing ratio precision
index int32 {time}   zero-based index of the sample within the source product

Mapping description

The table below details where and how each variable was retrieved from the input product.

field name mapping description

description the time converted from TAI93 to seconds since 2000-01-01T00:00:00

latitude path /HDFEOS/SWATHS/CO2NadirSwath/Geolocation_Fields/Latitude[]
altitude path /HDFEOS/SWATHS/CO2NadirSwath/Data_Fields/Altitude[]
pressure path /HDFEOS/SWATHS/CO2NadirSwath/Data_Fields/Pressure[]
