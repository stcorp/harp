S5P_L2_CLOUD

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of S5P_L2_CLOUD data.

field name type dimensions unit description
scanline_pixel_index int16 {time}   pixel index (0-based) within the scanline
datetime double {time} seconds since 2000-01-01 start time of the measurement
longitude float {time} degree_east longitude of the ground pixel center (WGS84)
latitude float {time} degree_north latitude of the ground pixel center (WGS84)
longitude_bounds float {time, 4} degree_east longitudes of the ground pixel corners (WGS84)
latitude_bounds float {time, 4} degree_north latitudes of the ground pixel corners (WGS84)
instrument_longitude float {time} degree_east longitude of the goedetic sub-instrument point (WGS84)
instrument_latitude float {time} degree_north latitude of the geodetic sub-instrument point (WGS84)
instrument_altitude float {time} m altitude of the instrument with respect to the geodetic sub-instrument point (WGS84)
solar_zenith_angle float {time} degree zenith angle of the Sun at the ground pixel location (WGS84); angle measured away from the vertical
solar_azimuth_angle float {time} degree azimuth angle of the Sun at the ground pixel location (WGS84); angle measured East-of-North
viewing_zenith_angle float {time} degree zenith angle of the instrument at the ground pixel location (WGS84); angle measured away from the vertical
viewing_azimuth_angle float {time} degree azimuth angle of the instrument at the ground pixel location (WGS84); angle measured East-of-North
cloud_fraction float {time}   retrieved fraction of horizontal area occupied by clouds using the OCRA/ROCINN CAL model
cloud_fraction_uncertainty float {time}   uncertainty of the retrieved fraction of horizontal area occupied by clouds using the OCRA/ROCINN CAL model
cloud_top_pressure float {time} hPa retrieved atmospheric pressure at the level of cloud using the OCRA/ROCINN CAL model
cloud_top_pressure_uncertainty float {time} hPa uncertainty of the retrieved atmospheric pressure at the level of cloud using the OCRA/ROCINN CAL model
cloud_top_height float {time} m retrieved vertical distance above the surface of the cloud top using the OCRA/ROCINN CAL model
cloud_top_height_uncertainty float {time} m uncertainty of the retrieved vertical distance above the surface of the cloud top using the OCRA/ROCINN CAL model
cloud_optical_depth float {time} m retrieved cloud optical depth using the OCRA/ROCINN CAL model
cloud_optical_depth_uncertainty float {time} m uncertainty of the retrieved cloud optical depth using the OCRA/ROCINN CAL model
index int32 {time}   zero-based index of the sample within the source product

Mapping description

The table below details where and how each variable was retrieved from the input product.

field name mapping description
scanline_pixel_index description the scanline and pixel dimensions are collapsed into a temporal dimension; the index of the pixel within the scanline is computed as the index on the temporal dimension modulo the number of scanlines
datetime path /PRODUCT/time, /PRODUCT/delta_time[]
description time converted from milliseconds since a reference time (given as seconds since 2010-01-01 UTC) to seconds since 2000-01-01 TAI; the time associated with a scanline is repeated for each pixel in the scanline
longitude path /PRODUCT/longitude[]
latitude path /PRODUCT/latitude[]
longitude_bounds path /PRODUCT/SUPPORT_DATA/GEOLOCATIONS/longitude_bounds[]
latitude_bounds path /PRODUCT/SUPPORT_DATA/GEOLOCATIONS/latitude_bounds[]
instrument_longitude path /PRODUCT/SUPPORT_DATA/GEOLOCATIONS/satellite_longitude[]
description the satellite longitude associated with a scanline is repeated for each pixel in the scanline
instrument_latitude path /PRODUCT/SUPPORT_DATA/GEOLOCATIONS/satellite_latitude[]
description the satellite latitude associated with a scanline is repeated for each pixel in the scanline
instrument_altitude path /PRODUCT/SUPPORT_DATA/GEOLOCATIONS/satellite_altitude[]
description the satellite altitude associated with a scanline is repeated for each pixel in the scanline
solar_zenith_angle path /PRODUCT/SUPPORT_DATA/GEOLOCATIONS/solar_zenith_angle[]
solar_azimuth_angle path /PRODUCT/SUPPORT_DATA/GEOLOCATIONS/solar_azimuth_angle[]
viewing_zenith_angle path /PRODUCT/SUPPORT_DATA/GEOLOCATIONS/viewing_zenith_angle[]
viewing_azimuth_angle path /PRODUCT/SUPPORT_DATA/GEOLOCATIONS/viewing_azimuth_angle[]
cloud_fraction path /PRODUCT/cloud_radiometric_fraction[]
cloud_fraction_uncertainty path /PRODUCT/cloud_radiometric_fraction_precision[]
cloud_top_pressure path /PRODUCT/cloud_top_pressure[]
cloud_top_pressure_uncertainty path /PRODUCT/cloud_top_pressure_precision[]
cloud_top_height path /PRODUCT/cloud_top_height[]
cloud_top_height_uncertainty path /PRODUCT/cloud_top_height_precision[]
cloud_optical_depth path /PRODUCT/cloud_optical_thickness[]
cloud_optical_depth_uncertainty path /PRODUCT/cloud_optical_thickness_precision[]
