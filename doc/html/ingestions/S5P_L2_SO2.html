Ingestion options

The table below lists the available ingestion options for S5P_L2_SO2 products.

option name

legal values

description

so2_column

1km, 7km, 15km

whether to ingest the SO2 column derived from the TM5 profile (default), from the 1km box profile (so2_column=1km), from the 7km box profile (so2_column=7km), or from the 15km box profile (so2_column=15km); providing this option will only work for NRTI data or for OFFL data with processor version >= 01.01.01 (otherwise an empty product is returned)

cloud_fraction

radiance

whether to ingest the cloud fraction (default) or the radiance cloud fraction (cloud_fraction=radiance)

description

pressure in Pa at tropause is derived from the upper bound of the layer with tropopause layer index k: exp((log(tm5_constant_a[k] + tm5_constant_b[k] * surface_pressure[]) + log(tm5_constant_a[k + 1] + tm5_constant_b[k + 1] * surface_pressure[]))/2.0)
