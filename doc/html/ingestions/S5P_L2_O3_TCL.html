Ingestion options

The table below lists the available ingestion options for S5P_L2_O3_TCL products.

option name

legal values

description

o3

ccd, csa

whether to ingest the tropical tropospheric ozone column with the Convective Cloud Differential (CCD) method (o3=ccd, default) or with the Cloud Slicing Algorithm (CSA) (o3=csa)

o3_strat

reference

whether to ingest the ozone_stratospheric_vertical_column (default), or ozone_stratospheric_vertical_column_reference (o3_strat=reference); this option only applies if CCD data is ingested
