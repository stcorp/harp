AEOLUS_L2B_Rayleigh

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of AEOLUS_L2B_Rayleigh data.

field name type dimensions unit description
datetime_start double {time} seconds since 2000-01-01 average datetime of the measurements used for the wind profile
latitude double {time} degree_north average latitude of the measurements used for the wind profile
longitude double {time} degree_east average longitude of the measurements used for the wind profile
altitude double {time, vertical} m vertical COG altitude relative to geoid for each accumulation
altitude_bounds double {time, vertical, 2} m altitude relative to geoid of layer boundaries for each accumulation
hlos_wind_velocity double {time, vertical} cm/s HLOS wind velocity
hlos_wind_velocity_validity int8 {time, vertical}   validity flag of the HLOS wind velocity
index int32 {time}   zero-based index of the sample within the source product

Ingestion options

The table below lists the available ingestion options for AEOLUS_L2B products.

option name legal values description
data rayleigh, mie the type of profiles to ingest (rayleigh/mie)

This definition is only applicable when: data=rayleigh

Mapping description

The table below details where and how each variable was retrieved from the input product.

field name mapping description
datetime_start path /rayleigh_profile[]/profile_datetime_average
latitude path /rayleigh_profile[]/profile_lat_average
longitude path /rayleigh_profile[]/profile_lon_average
altitude path /rayleigh_geolocation[/rayleigh_profile/l2b_wind_profies/wind_result_id_number - 1]/windresult_geolocation/altitude_vcog
altitude_bounds path /rayleigh_geolocation[/rayleigh_profile/l2b_wind_profies/wind_result_id_number - 1]/windresult_geolocation/altitude_bottom, /rayleigh_geolocation[/rayleigh_profile/l2b_wind_profies/wind_result_id_number - 1]/windresult_geolocation/altitude_top
hlos_wind_velocity path /rayleigh_hloswind[/rayleigh_profile/l2b_wind_profies/wind_result_id_number - 1]/windresult/rayleigh_wind_velocity
hlos_wind_velocity_validity path /rayleigh_hloswind[/rayleigh_profile/l2b_wind_profies/wind_result_id_number - 1]/windresult/validity_flag
