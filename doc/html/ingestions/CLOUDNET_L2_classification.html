CLOUDNET_L2_classification

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of CLOUDNET_L2_classification data.

field name type dimensions unit description
datetime double {time} seconds since 2000-01-01 date and time (in seconds since 2000-01-01 00:00:00)
sensor_latitude double   degree_north latitude of the instrument
sensor_longitude double   degree_east longitude of the instrument
sensor_altitude double   km altitude of the instrument above mean sea level
cloud_base_height double {time} km cloud_base_height
cloud_top_height double {time} km cloud_top_height
index int32 {time}   zero-based index of the sample within the source product

Mapping description

The table below details where and how each variable was retrieved from the input product.

field name mapping description
datetime path /time
description hours since 00:00:00 of the day of the measurement
sensor_latitude path /latitude
sensor_longitude path /longitude
sensor_altitude path /altitude
cloud_base_height path /cloud_base_height
cloud_top_height path /cloud_top_height
