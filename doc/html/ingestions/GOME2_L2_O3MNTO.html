Ingestion options

The table below lists the available ingestion options for GOME2_L2_O3MNTO products.

option name

legal values

description

detailed_results

BrO, H2O, HCHO, NO2, O3, OClO, SO2

include additional detailed results for the given species

corrected

true

include corrected VCD and/or ESC (corrected=true) or uncorrected VCD/ESC (default); this only applies to the species for which additional detailed results are ingested (detailed_results is set to one of BrO, H2O, HCHO, NO2, O3, SO2, or OClO); for SO2 this option is only applicable for the 6km height (other heights always provide corrected columns)

so2_column

15km, 6km, 2.5km, 1km

plume height retrieval version of the SO2 column to ingest (default is 6km); only applicable if detailed_results is set to SO2
